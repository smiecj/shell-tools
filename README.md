# shell tools
some shell for install and manage components

most shell come from [docker-centos](https://github.com/smiecj/docker-centos)

[中文](https://github.com/smiecj/shell-tools/blob/main/README_zh.md)

## python

install conda & python3

```
make python3
```

## nodejs

install nodejs

```
make nodejs
```

## java

install java (1.8 & 17)

```
make java
```

## golang

install golang

```
make golang
```
